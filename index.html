LangChain & OpenAI Research Paper Assistant

Link to Project

This is a project I built for a Streamlit + LangChain Intro Workshop I led in Purdue's InnovateHer Hackathon (focused on solving women's issues). The application is a research assistant chatbot using Streamlit and LangChain, integrating OpenAI's API for NLP. The workshop includes everything from setting up the development environment and developing Python methods in the Streamlit app that extract and process PDF content. The result is users are able to upload a PDF of a research paper and ask the chatbot questions about it to receive structured responses as well as chat history storage.
